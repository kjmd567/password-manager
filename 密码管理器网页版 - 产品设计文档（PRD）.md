# 密码管理器网页版 - 产品设计文档（PRD）

**适用**：交给 AI 直接编程实现
**风格**：极简、清晰、无歧义、纯前端可直接落地
**核心确定**：Vue3 前端框架 + 干净简约+液态玻璃设计风格 + 手机端/网页端双适配 + 简单AES加密 + 电脑/手机作为默认账号
**技术选型**：聚焦Vue3生态，轻量易实现，贴合液态玻璃风格与多端适配需求，具体如下

# 一、产品定位

一款**本地优先、轻量安全、多端适配（手机端+网页端同等适配）**的密码管理器，核心设定：以用户使用的电脑/手机作为默认账号，无需额外注册，仅需输入密码验证；采用Vue3开发，视觉上遵循干净简约+液态玻璃设计风格，用于：

- 存储各类网站/软件的账号、密码、备注

- 快速搜索、复制密码，适配手机端触控操作与网页端便捷操作

- 本地加密存储（简单AES），不上传云端，保障数据隐私

- 支持导出/导入备份，适配多端操作习惯，视觉体验统一

# 二、核心功能（必做，含Vue3适配与风格适配）

## 1. 登录/解锁页面（核心调整）

核心逻辑：**以当前使用的电脑/手机作为默认账号**，无需输入账号，仅需验证密码，简化登录流程；视觉贴合液态玻璃风格，简洁无多余元素，同时保障安全性

- 仅需输入主密码（本地验证，与当前设备绑定，换设备需重新设置主密码/导入备份）

- 记住解锁状态（可选，仅保存在当前设备的浏览器本地存储，换设备无效）

- 连续错误输入临时锁定（防暴力尝试，如连续5次错误，锁定10分钟）

- 首次使用引导：打开工具后，直接提示“设置主密码（当前设备为默认账号）”，无需注册，引导界面简洁，贴合液态玻璃风格

- 多端适配：手机端、网页端界面统一，按钮、输入框尺寸适配对应设备，触控/点击体验流畅

## 2. 密码列表页（主页，多端适配+风格适配）

- 布局适配：网页端用卡片式布局，手机端用列表式布局，均融入液态玻璃效果（半透明、模糊、轻微边框发光），干净简约无多余装饰

- 每条显示：网站名/账号/创建时间/标签，手机端可左右滑动查看更多信息，网页端悬浮显示详情，视觉统一

- 搜索：按网站名、账号、备注模糊搜索，手机端搜索框固定在顶部，网页端搜索框悬浮顶部（贴合液态玻璃），方便快速查找

- 排序：按创建时间、名称、使用频率，支持一键切换，切换动画简洁流畅，符合简约风格

- 快捷操作：网页端点击/悬浮触发操作，手机端长按弹出操作菜单，操作按钮融入液态玻璃质感，点击有轻微反馈

## 3. 新增/编辑密码

**字段**：

- 网站/软件名称（必填）

- 账号（必填）

- 密码（必填）

- 网址（可选）

- 备注（可选）

- 标签（可选，如：工作、社交、支付，方便分类）

**功能**：

- 密码可见/隐藏切换（点击小眼睛图标，手机端点击适配触控尺寸，网页端图标贴合液态玻璃风格）

- 一键复制账号、密码，复制后弹出“已复制”提示（网页端悬浮液态玻璃提示，手机端底部简约提示）

- 随机密码生成器（支持一键填充到密码字段，生成器界面简洁，按钮融入液态玻璃质感）

## 4. 密码生成器

可配置（手机端用滑动条/勾选框，网页端用勾选框+输入框，操作便捷，界面简约）：

- 长度（8–32位，默认16位）

- 大写字母（默认勾选）

- 小写字母（默认勾选）

- 数字（默认勾选）

- 特殊符号（可选，如!@#$%^&*）

- 一键复制生成的密码，支持直接粘贴到新增/编辑页面，复制按钮贴合液态玻璃风格

## 5. 数据安全与本地存储（与设备账号绑定，简单AES加密）

- 所有数据**仅存在当前设备（电脑/手机）的本地存储**（电脑端用localStorage，手机端用浏览器内置localStorage，Vue3统一调用本地存储API）

- 数据用**简单AES对称加密**后再存储，加密密钥与当前设备绑定，换设备无法直接解密，无需复杂加密逻辑

- 退出/锁定后，立即清除内存中的明文数据，防止他人查看，Vue3中通过销毁组件、清空响应式数据实现

- 无后台、无服务器、不上传任何数据，所有操作均在本地完成，Vue3纯前端实现

- 安全提示：换设备时，需通过“导出备份”功能，将加密数据转移到新设备，再输入主密码解密，提示界面简约，贴合整体风格

## 6. 数据备份与恢复

- 导出：将加密后的密码数据，导出为加密文件（.json 或 .pwm 格式），保存到本地，导出按钮融入液态玻璃质感

- 导入：仅支持本工具导出的加密文件，导入时需输入主密码验证，防止非法导入，验证弹窗简约，贴合液态玻璃风格

- 导入前二次确认，明确提示“导入将覆盖当前设备的所有密码数据”，避免误操作

## 7. 设置页

- 修改主密码（需输入原密码验证，修改后立即重新加密所有数据，表单界面简约，贴合液态玻璃风格）

- 自动锁定时间（可选：1/5/15/30分钟，无操作时自动锁定，返回解锁页，选项样式统一，简洁易选）

- 清空所有数据（危险操作，二次确认，清空后无法恢复，确认弹窗简约，突出风险提示）

- 多端适配设置（可选：手机端字体大小调整、网页端主题切换，切换控件贴合液态玻璃风格）

- 关于/版本/帮助（简单说明“电脑/手机作为默认账号”的逻辑，解答常见问题，界面干净无多余元素）

# 三、非功能要求（AI 必须实现，贴合Vue3与风格需求）

1. **多端适配**：手机端、网页端同等适配，Vue3+响应式布局实现，电脑端适配Chrome 100+、Edge 100+、Firefox 95+，手机端适配微信浏览器、Chrome手机版、Safari手机版（iOS）、华为浏览器，触控/点击操作流畅

3. **加密**：使用简单AES对称加密，调用CryptoJS前端加密库，Vue3中封装统一加密/解密方法，无需自定义加密算法，逻辑简单可落地

4. **安全**：密码不明文显示在控制台、不明文缓存，解锁状态下，切换浏览器标签/最小化，不自动暴露密码，Vue3中做好数据脱敏处理

5. **视觉风格**：严格遵循干净简约+液态玻璃设计风格，核心视觉特点：半透明界面、轻微模糊效果、淡色边框发光、低饱和度配色、无多余装饰，整体简洁统一

# 七、核心流程（AI 必须按流程写逻辑，Vue3适配，含设备绑定）

## 1. 首次使用（新设备）

打开工具 → Vue3初始化 → 调用封装方法获取当前设备标识（电脑/手机）→ 提示“设置主密码（当前设备为默认账号）”→ 确认主密码 → 调用简单AES加密方法，加密存储主密码哈希值与设备标识 → 跳转至主页（Vue3路由跳转）

## 2. 再次打开（同一设备）

打开工具 → Vue3初始化 → 检测到当前设备标识已绑定主密码 → 显示密码输入框 → 输入密码 → 调用加密方法验证主密码哈希值 → 解密本地数据 → 显示密码列表（若勾选“记住解锁状态”，则直接解锁，Vue3响应式渲染数据）

## 3. 换设备使用

原设备导出加密备份文件 → 新设备打开工具 → 首次使用设置新主密码 → 导入备份文件 → 输入原设备主密码 → 调用简单AES解密方法解密数据 → 绑定新设备标识 → 正常使用

## 4. 新增密码

点击“新增”（Vue3路由跳转至/add）→ 填写各项信息 → （可选）使用密码生成器生成密码 → 点击保存 → 调用简单AES加密方法，加密数据 → 存储到本地 → 跳转回主页，刷新列表（Vue3响应式更新）

## 5. 查看/复制密码

进入密码列表 → 找到目标项 → 点击小眼睛查看密码（10秒后自动隐藏，Vue3定时器控制）→ 点击复制按钮 → 调用clipboard API复制 → 弹出“已复制”提示

## 6. 退出/锁定

点击“锁定”/无操作达到自动锁定时间 → Vue3销毁当前页面组件、清空响应式明文数据 → 销毁当前解锁状态 → 路由跳转至解锁页

## 7. 导出/导入备份

导出：进入设置 → 点击“导出备份” → 调用加密方法生成加密文件 → 调用File API下载到本地；导入：进入设置 → 点击“导入备份” → 调用File API读取文件 → 输入原主密码 → 验证通过 → 覆盖当前数据并加密存储 → 响应式刷新列表

# 八、技术选型（Vue3为主，AI 可直接落地，贴合需求）

核心原则：Vue3轻量开发、无后端依赖、多端适配、安全易实现，贴合干净简约+液态玻璃风格，所有技术均选用主流、轻量、现成组件，降低AI编程难度。

## 1. 基础前端框架

- 核心框架：**Vue3（轻量版，无需Vuex/Pinia）**，使用Vue3核心语法（setup语法糖、响应式数据、组件复用），无需复杂状态管理，简化开发

- 路由管理：**Vue Router 4**，实现文档中所有路由跳转，配置简洁，适配多端跳转逻辑

- 优势：组件化开发，表单处理、页面跳转、组件复用更便捷，适配多端响应式，AI 易生成代码，可快速实现液态玻璃风格界面

## 2. 样式与响应式适配（贴合液态玻璃风格）

- 基础样式：CSS3（Flex/Grid 布局）+ Vue3 内联样式/样式模块，实现电脑端卡片、手机端列表的适配布局，重点实现液态玻璃效果（半透明、模糊、发光）

- 响应式适配：Media Query（媒体查询）+ Vue3 响应式API（如ref、computed），自动适配电脑端（≥1200px）、手机端（≤768px）屏幕尺寸

- 简化开发：引入 **Tailwind CSS 轻量版**，快速实现响应式布局与液态玻璃样式，无需手写复杂CSS，贴合干净简约风格

## 3. 数据存储与设备标识获取

- 本地存储：电脑端、手机端统一使用 **localStorage**，Vue3中封装统一的存储/读取方法，简化操作

- 设备标识获取：前端通过 **navigator API** 自动获取设备/浏览器唯一标识（无需用户输入），Vue3中封装获取方法，推荐方案：navigator.userAgent + navigator.platform 组合，生成设备唯一标识（简单易实现，适配所有浏览器）

## 4. 加密技术（简单AES，贴合需求）

- 核心加密：**简单AES 对称加密**，使用现成前端加密库，无需自定义加密算法

- 推荐库：CryptoJS（轻量、主流，AI 易调用，支持AES加密/解密），Vue3中封装统一的加密、解密方法，供全项目调用

- 加密逻辑：主密码 + 设备标识组合作为加密密钥，对密码数据（pwm_data）进行加密，对主密码进行哈希处理（存储pwm_master_hash），逻辑简单，无需复杂配置

## 5. 核心功能技术实现（Vue3适配，对应文档功能）

- 密码生成器：Vue3 自定义函数，根据配置（长度、字符类型）随机生成密码，支持一键复制（调用 clipboard API，Vue3中封装复制方法）

- 搜索与排序：Vue3 响应式数据 + JavaScript 数组方法（filter、sort），实现模糊搜索、多条件排序，无需复杂逻辑，实时响应搜索结果

- 导入/导出：使用 File API 实现本地文件读取（导入）、文件下载（导出），Vue3中绑定按钮点击事件，实现对应功能，仅支持本工具导出的加密文件

- 自动锁定：Vue3 定时器（setTimeout）+ 事件监听（监听页面点击、滚动等操作），达到设定时间自动锁定，销毁响应式数据

- 弹窗与提示：使用 Vue3 自定义弹窗组件（贴合液态玻璃风格），替代原生弹窗，实现二次确认、复制提示等功能，组件可复用

## 6. 兼容性要求（AI 必须满足）
