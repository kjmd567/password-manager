# 密码管理器使用指南

## 一、基本使用

### 首次使用
1. **打开应用**：在浏览器中打开密码管理器
2. **设置主密码**：首次使用时，系统会提示您设置主密码（当前设备将作为默认账号）
3. **确认主密码**：再次输入主密码进行确认
4. **进入主界面**：设置完成后自动进入密码列表页

### 日常使用

#### 添加密码
1. 点击右下角的"+"按钮
2. 填写网站名称、账号、密码等信息
3. 可选择添加标签、备注和网址
4. 点击"保存"完成添加

#### 查看/复制密码
1. 在密码列表中找到需要的密码项
2. 点击小眼睛图标可查看密码（10秒后自动隐藏）
3. 点击复制按钮可将账号或密码复制到剪贴板

#### 编辑密码
1. 点击密码卡片上的"编辑"按钮
2. 修改相关信息后点击"保存"

#### 删除密码
1. 点击密码卡片上的"删除"按钮
2. 确认删除操作

#### 使用密码生成器
1. 点击"生成密码"按钮进入密码生成器
2. 调整密码长度和字符类型
3. 点击"复制"按钮复制生成的密码
4. 点击"使用此密码"返回添加页面并自动填充

## 二、数据管理

### 导出备份
1. 进入设置页面
2. 点击"导出"按钮
3. 下载加密的备份文件到本地

### 导入备份
1. 进入设置页面
2. 点击"导入"按钮
3. 选择之前导出的备份文件
4. 输入主密码进行验证
5. 确认导入操作

### 清空数据
1. 进入设置页面
2. 点击"清空"按钮
3. 确认清空操作（此操作不可恢复）

## 三、安全设置

### 修改主密码
1. 进入设置页面
2. 点击"修改"按钮
3. 输入原密码和新密码
4. 确认修改

### 设置自动锁定时间
1. 进入设置页面
2. 在"自动锁定时间"下拉菜单中选择合适的时间
3. 系统会在无操作达到设定时间后自动锁定

## 四、常见问题与解决方法

### 1. 忘记主密码
**问题**：无法解锁密码管理器
**解决方法**：
- 如果有备份文件，可以在新设备上重新设置主密码并导入备份
- 如果没有备份，无法恢复数据，需要重新开始使用

### 2. 换设备后数据丢失
**问题**：在新设备上打开应用后没有之前的密码数据
**解决方法**：
- 在原设备上导出备份文件
- 在新设备上设置新主密码
- 导入备份文件并输入原设备的主密码

### 3. 浏览器清除数据后密码丢失
**问题**：浏览器清除缓存或本地存储后，密码数据丢失
**解决方法**：
- 定期导出备份文件，存放在安全的地方
- 不要随意清除浏览器的本地存储数据

### 4. 密码不显示或无法复制
**问题**：点击小眼睛图标后密码不显示，或复制按钮不工作
**解决方法**：
- 检查浏览器是否允许JavaScript
- 确保浏览器支持Clipboard API
- 尝试刷新页面

### 5. 导入备份失败
**问题**：导入备份文件时提示密码错误或文件格式不正确
**解决方法**：
- 确保输入的是原设备的主密码
- 确保备份文件没有损坏
- 确保备份文件是由本密码管理器导出的

## 五、注意事项

1. **数据安全**：所有数据仅存储在本地，不上传云端，请定期导出备份
2. **主密码**：主密码是唯一的解锁方式，请妥善保管
3. **设备绑定**：密码数据与当前设备绑定，换设备需要通过导出/导入迁移数据
4. **浏览器兼容性**：建议使用Chrome、Edge、Firefox等现代浏览器
5. **网络安全**：使用HTTPS连接访问密码管理器，避免在公共网络上使用

## 六、快捷键

- **添加密码**：无快捷键，点击"+"按钮
- **搜索**：在搜索框中输入关键词
- **复制密码**：点击密码旁的复制按钮
- **查看密码**：点击小眼睛图标
