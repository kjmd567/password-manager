# 密码管理器技术文档

## 一、技术栈

### 前端框架
- **Vue 3**：使用Vue 3的组合式API，采用setup语法糖，简化代码结构
- **Vue Router 4**：实现页面路由管理，支持多页面切换

### 构建工具
- **Vite**：现代化前端构建工具，提供快速的开发体验和优化的生产构建

### 加密技术
- **CryptoJS**：用于实现AES对称加密，保护密码数据安全

### 存储方案
- **localStorage**：用于本地存储加密后的密码数据和用户设置

### 样式实现
- **CSS3**：使用Flexbox和Grid布局，实现响应式设计
- **媒体查询**：适配不同屏幕尺寸，实现移动端和桌面端的响应式布局
- **CSS变量**：便于统一管理颜色和样式

## 二、核心技术实现

### 1. 设备标识与账号管理

**实现方式**：
- 通过`navigator.userAgent`和`navigator.platform`组合生成设备唯一标识
- 将设备标识存储在localStorage中，作为默认账号
- 无需用户手动输入账号，简化登录流程

**代码位置**：`src/utils/crypto.js`中的`getDeviceId`函数

### 2. 加密与解密

**实现方式**：
- 使用CryptoJS库实现AES对称加密
- 主密码与设备标识组合生成加密密钥
- 对密码数据进行加密后存储，确保数据安全
- 验证主密码时使用哈希比对，不存储明文密码

**代码位置**：`src/utils/crypto.js`中的加密和解密函数

### 3. 本地存储管理

**实现方式**：
- 封装localStorage操作，提供统一的存储和读取方法
- 存储结构包括：设备标识、主密码哈希、加密的密码数据、记住状态、自动锁定时间
- 提供导出/导入功能，支持数据备份和迁移

**代码位置**：`src/utils/storage.js`

### 4. 响应式设计

**实现方式**：
- 使用CSS媒体查询适配不同屏幕尺寸
- 桌面端采用卡片式布局，移动端采用列表式布局
- 按钮和输入框尺寸根据设备类型自动调整
- 导航和操作方式适配触控和鼠标操作

**代码位置**：各组件的`<style scoped>`部分

### 5. 液态玻璃风格

**实现方式**：
- 使用`backdrop-filter: blur()`实现模糊效果
- 半透明背景色增强层次感
- 轻微的边框发光效果提升视觉体验
- 悬停动画增强交互反馈

**代码位置**：各组件的样式部分

### 6. 自动锁定机制

**实现方式**：
- 使用`setTimeout`实现无操作自动锁定
- 监听用户操作（鼠标移动、键盘输入、点击）重置锁定计时器
- 锁定后清空内存中的明文数据，跳转到登录页

**代码位置**：`src/views/Home.vue`中的`resetLockTimer`函数

### 7. 密码生成器

**实现方式**：
- 根据用户配置的长度和字符类型生成随机密码
- 支持大写字母、小写字母、数字和特殊符号
- 一键复制生成的密码到剪贴板
- 与添加密码页面集成，支持直接使用生成的密码

**代码位置**：`src/views/PasswordGenerator.vue`

## 四、关键功能实现细节

### 1. 登录/解锁流程

1. **首次使用**：
   - 检测localStorage中是否存在主密码哈希
   - 不存在则显示设置主密码界面
   - 设置主密码后，生成设备标识并存储

2. **再次使用**：
   - 检测设备标识和主密码哈希
   - 显示密码输入界面
   - 验证密码正确性后解锁
   - 支持记住解锁状态

### 2. 密码管理

1. **添加密码**：
   - 收集用户输入的密码信息
   - 生成唯一ID
   - 加密后存储到localStorage

2. **编辑密码**：
   - 根据ID查找密码项
   - 预填充表单
   - 更新后重新加密存储

3. **删除密码**：
   - 根据ID从列表中移除
   - 重新加密存储

4. **搜索与排序**：
   - 实时过滤密码列表
   - 支持按创建时间和名称排序

### 3. 数据备份与恢复

1. **导出备份**：
   - 读取加密的密码数据
   - 生成JSON格式的备份文件
   - 下载到本地

2. **导入备份**：
   - 读取本地备份文件
   - 验证主密码
   - 解密并存储数据

### 4. 安全措施

1. **数据加密**：
   - 所有密码数据使用AES加密
   - 主密码使用哈希存储

2. **防暴力破解**：
   - 连续错误输入5次后锁定10分钟

3. **自动锁定**：
   - 无操作时自动锁定
   - 锁定后清空内存数据

4. **数据隔离**：
   - 不同设备的数据相互隔离
   - 换设备需要重新设置或导入

## 五、性能优化

1. **代码分割**：
   - 使用Vue Router的懒加载功能
   - 减少初始加载时间

2. **本地存储优化**：
   - 仅存储必要的数据
   - 加密后存储，减少存储空间

3. **响应式设计**：

   - 减少不必要的DOM操作

4. **缓存策略**：

本密码管理器采用Vue 3 + Vite + CryptoJS技术栈，实现了一个本地优先、安全可靠的密码管理工具。通过设备标识作为默认账号，简化了用户登录流程；使用AES加密保护数据安全；采用响应式设计适配多端；实现了完整的密码管理、数据备份和安全设置功能。
